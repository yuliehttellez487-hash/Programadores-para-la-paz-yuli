Incidencia 1:
Título: Error 500 al enviar datos a /incidencia
Descripción: El servidor no reconoce req.body.tipo porque falta app.use(express.json())
Pasos para reproducir:
1. Abrir Postman
2. Enviar POST a http://localhost:3000/incidencia 
3. Mandar JSON { "tipo": "reporte", "descripcion": "prueba" }
Resultado esperado: Recibir respuesta con "estado": "Datos recibidos"
Resultado real: Servidor devuelve error 500 y tipo = undefined
Solución: Agregar app.use(express.json()) antes de las rutas para que Express lea el JSON
